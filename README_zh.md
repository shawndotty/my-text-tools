# Obsidian 文本工具箱 (My Text Tools)

一个受 [mytexttools.com](https://mytexttools.com) 启发并致敬的高级 Obsidian 文本处理工作台。

本插件为 Obsidian 增加了一个专门的“文本工具工作台”，提供了一套强大的工具，让您无需离开笔记库即可对文本进行操作、格式化和分析。它拥有一个临时的编辑区域，允许您在应用更改到笔记之前安全地处理文本。

## 功能特性

My Text Tools 将其实用工具分为以下几类：

### 🛠️ 基础工具

-   **正则查找与替换**：使用正则表达式执行复杂的搜索和替换操作。
-   **去除空白字符**：
    -   去除首尾空白。
    -   将多个连续空格压缩为一个。
    -   删除所有空格或制表符。

### 📝 行处理工具

-   **去除重复行**：通过删除重复项快速清理列表。
-   **去除空行**：从文本中剥离空行。
-   **添加前缀/后缀**：批量向每一行的开头或结尾添加文本。
-   **过滤行**：根据是否包含特定字符串或正则模式来保留或删除行。
-   **列表编号**：将行转换为有序列表，支持自定义起始数字、步长和格式。
-   **换行处理**：管理换行符（添加或删除）。

### 📊 列处理工具

非常适合处理类 CSV 数据或结构化文本：

-   **提取列**：根据分隔符（逗号、制表符、竖线、空格或自定义）提取特定列。
-   **交换列**：轻松交换两列的位置。

### 🔍 提取与分析

-   **内容提取**：提取两个标记（开始/结束标签）之间的文本，支持正则。
-   **词频分析**：分析文本以查找出现频率最高的单词，支持按长度过滤及包含/排除数字。

## 使用指南

1. **打开工作台**：

    - 点击左侧侧边栏的“My Text Tools”图标（清除格式图标）。
    - 或者使用命令面板 (`Cmd/Ctrl + P`) 并搜索 **"My Text Tools: 开启 MyTextTools 增强工作台"**。

2. **工作台界面**：

    - **左侧面板**：选择您想要使用的工具。
    - **中间面板（临时编辑器）**：
        - 此文本区域初始化为您当前活动笔记的内容。
        - **此处的修改是临时的**，直到您选择保存才会影响您的笔记。
        - 支持 **撤销 (Ctrl+Z)** 和 **重做 (Ctrl+Y)**。
    - **右侧面板**：配置所选工具的设置（例如，正则模式、分隔符）。

3. **保存更改**：
    - **应用并覆盖原笔记**：使用工作台的内容覆盖原始笔记。
    - **保存为新笔记**：创建一个包含处理后文本的新文件。

## 安装方法

目前，本插件支持手动安装：

1. 从最新发布页面下载 `main.js`、`manifest.json` 和 `styles.css` 文件。
2. 在您的笔记库插件目录中创建一个名为 `my-text-tools` 的文件夹：`.obsidian/plugins/my-text-tools`。
3. 将下载的文件放入此文件夹中。
4. 重新加载 Obsidian 并在“设置 > 第三方插件”中启用该插件。
